# MusicApp

A minimal, high-fidelity music player for macOS — built entirely in SwiftUI.

## Features

- **Album Carousel** — browse albums with smooth 3D-tilt interactions and animated covers
- **Waveform Scrubber** — visual waveform display with drag-to-seek
- **Queue Management** — drag-to-reorder, play next, and loop modes
- **Media Keys** — play/pause, next/prev via keyboard and system controls
- **AI Album Art Animations** — Google Gemini-powered Ken Burns, parallax, and ambient glow effects
- **Rich Metadata** — disc/track numbers, sample rate, bit depth, codec info, Apple Digital Master detection
- **Full-Screen Support** — dynamic layout adapts to any window size

## Requirements

- macOS 13.0+
- Xcode 15+

## Building

1. Clone this repository
2. Open `MusicApp.xcodeproj` in Xcode
3. Build & Run (⌘R)

## Usage

1. Click the **folder icon** in the mini player to select a directory of music files
2. Supported formats: AAC, ALAC, MP3, FLAC, WAV
3. Use ⌘← / ⌘→ for previous/next track, Space for play/pause
